UserProfile

final class UserProfile : Codable
extension UserProfile: PostgreSQLUUIDModel
extension UserProfile: Content
extension UserProfile: Parameter
extension UserProfile: Migration

The optional information provided by a user for their profile page. The associated UserProfile is automatically created wnen a User is created via the endpoint. Only the .userID, .username and .userSearch fields are populated upon initialization.
